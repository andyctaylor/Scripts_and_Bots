Email Template Generator
A Python project for generating personalized email templates using HTML and SMTP.

Features

HTML Templating: Create reusable, well-structured email templates with HTML.
Variable Substitution: Personalize your emails with dynamic content (e.g., recipient's name).
SMTP Integration: Send generated emails via the SMTP protocol.

Usage

Create an HTML template (index.html):

HTML
<!DOCTYPE html>
<html>
<body>
<h1>Hello, $name!</h1>
<p>This is a test email generated by my Python script.</p>
</body>
</html>

Write a Python script:

Python
import smtplib
from email.message import EmailMessage
from string import Template
from pathlib import Path

html = Template(Path('index.html').read_text())
email_message = EmailMessage()

email_message['To'] = 'andyt@plusultras.org'
email_message['From'] = 'Angel Taylor'
email_message['Subject'] = 'This is me testing my email object'

email_message.set_content(html.substitute(name='Your Name'), 'html')

with smtplib.SMTP(host='smtp.gmail.com', port=587) as smtp:
    smtp.ehlo()
    smtp.starttls()
    smtp.login('Your_email_address', 'Your_email_password')
    smtp.send_message(email_message)
    print('Message has been sent.')
